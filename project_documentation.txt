项目文档 - ClosetAI

## 项目概述
ClosetAI是一个智能衣橱管理应用，帮助用户记录衣物、管理穿搭，并提供AI建议。

## 文件结构详解

### 前端页面文件 (frontend/)

#### index.html - 首页
- 显示用户的衣橱统计数据
- 展示推荐搭配和穿搭记录
- 提供快速访问其他功能的入口

#### wardrobe.html - 衣橱页面
- 显示用户的所有衣物
- 支持按分类筛选
- 提供添加和删除衣物的功能
- 实现了拖拽上传功能

#### outfit.html - 穿搭页面
- 显示每日穿搭推荐
- 集成天气API，根据天气情况推荐合适的穿搭
- 支持用户自主选择衣物搭配

#### suggestion.html - 建议页面
- 提供AI穿搭建议
- 支持按风格和场合筛选建议
- 展示搭配技巧和流行趋势

#### tryon.html - 试衣间页面（新增）
- 实现了3D模特试衣功能
- 支持男女性别切换
- 使用Three.js加载FBX格式的3D模型
- 提供简单的衣物搭配功能

### 前端资源文件

#### style.css - 全局样式
- 定义了应用的色彩系统和排版
- 包含响应式设计的样式规则
- 实现了统一的卡片、按钮和表单样式

#### app.js - 应用脚本
- 处理页面间的导航和交互
- 实现了衣物的添加、删除和更新功能
- 管理用户的数据存储和本地状态

#### storage.js - 本地存储脚本
- 封装了对localStorage的操作
- 提供了衣橱数据的CRUD方法
- 实现了数据的序列化和反序列化

#### assets/models/ - 3D模型文件
- male.fbx: 男性3D模型
- female.fbx: 女性3D模型

### 后端文件 (backend/)

#### main.py - 后端API
- 使用FastAPI构建
- 提供衣物识别和AI建议功能
- 集成了Qwen API（阿里云智能对话）
- 实现了图片上传和分析接口

#### requirements.txt - Python依赖
- 列出了后端需要的所有Python包

### 测试文件

#### test_styles.py - 样式测试
- 验证页面样式是否正确加载
- 检查页面元素是否可见

#### test_suggestion.py - 建议页面测试
- 测试建议页面的功能
- 验证AI建议是否能正常生成

#### test_tryon.py - 试衣间页面测试
- 测试3D模型加载功能
- 验证性别切换和交互功能

#### test_wardrobe.py - 衣橱页面测试
- 测试衣橱页面的衣物管理功能
- 验证筛选和搜索功能

### 开发工具

#### convert_fbx_to_glb.py - 模型转换工具
- 使用Blender将FBX格式模型转换为GLB格式
- 提供命令行接口，方便批量处理

## 技术栈

### 前端
- HTML5 + CSS3 + JavaScript (ES6+)
- Three.js - 3D渲染库
- Font Awesome - 图标库
- Google Fonts - 字体资源

### 后端
- Python 3.9+
- FastAPI - Web框架
- Uvicorn - ASGI服务器
- Qwen API - 阿里云智能对话

### 测试
- Playwright - 浏览器自动化测试
- Pytest - Python单元测试

## 部署说明

### 开发环境
1. 安装Python依赖：`pip install -r backend/requirements.txt`
2. 启动后端服务：`cd backend && python -m uvicorn main:app --reload --port 8001`
3. 在浏览器中访问：http://localhost:8001

### 生产环境
1. 构建前端资源（如需）
2. 配置生产服务器
3. 设置环境变量
4. 启动后端服务

## 功能概述

### 用户功能
- 衣物管理：添加、删除、筛选、搜索
- 穿搭记录：记录每日穿搭
- AI建议：获取智能搭配建议
- 试衣间：3D模型试衣功能
- 数据统计：查看穿搭统计

### 技术功能
- 图片识别：AI识别衣物类型和颜色
- 天气集成：根据天气推荐穿搭
- 本地存储：数据保存在浏览器本地
- 响应式设计：支持移动设备
